# アプリ名
クイズゲーム

# 概要

youtubeの解説動画を参考にDBを使わずjavascriptとhtmlだけで作った

選択式のクイズゲームです

回答のボタンをクリックすると正解/不正解が表示され、

最後の問題が終了した後に合計正解数が表示されます(現在３問)
